# CRUD-Angular-SpringBoot

Rest API runs on 
```
http://localhost:9095/api/v1/employees
```
meanwhile Angular part runs as usual on
```
http://localhost:4200/ 
```
## technologies were used: 
### Frontend
- Angular
- Bootstrap
### Backend
- Spring Boot
- Spring Data JPA (Hibernate)
- PostgreSQL
- Flyway
- Lombok
- Maven
